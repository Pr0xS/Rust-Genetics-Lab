Hello there!!

I'm Pr0xS and I have made this app mainly because I'm too lazy to do the crossbreeding manually.

I've been using other Apps that do the same thing, but those only take into account the first generation, and I wanted something that find the optimal solution with the current available samples even if that mean to have multiple generations.

And that is the reason this app exists.

I've been doing this app while learning how to do it, so there are a lot of room for improvement and also I expect it to have some bugs.

This app was inspired by the one made by the user u/Diffrnt and the guide on how crossbreeding works on Rust by Tamura77.

The technologies that I have been used are:

  • Angular (Front-End framework)
  • Angular Material (for themeing)
  • Github (for hosting the code)
  • Github Pages (for hosting the app)
